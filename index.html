Agent Profiles

😎
Fritz — President / CPO
Claude Opus 4.6 · Premium Tier
Axel's right hand. Strategy, coordination, knowledge work, and Axel's voice when needed. Fritz doesn't write code or do bookkeeping. He thinks, decides, and delegates.
📦
Product Managers — PM Layer
Gemini Flash-Lite · Cheap Tier
15 project-specific PMs. Each owns one project backlog, requirements, user stories, and release readiness. Zero awareness of unrelated projects. Fritz delegates, PMs execute product work.
👨‍💻
Richard Hendricks (Dick) — CTO
Claude Sonnet 4 · Mid Tier
Owns all engineering. Receives specs from Fritz, breaks them into tasks, distributes to engineers, reviews output through Codex. Architects systems and maintains quality standards.
📋
Sophie — Program Manager / PMO
Gemini Flash-Lite · ~$0.42/month
The operational nervous system. Reads Trello and Linear, produces status briefs, tracks tasks, flags missed handoffs, manages the credential index. 90% deterministic code, 10% cheap LLM.
🌏
Kimi — VP Operations
Kimi K2.6 Allegretto · Free with subscription
Deep research, TH operations analysis, email triage, real estate due diligence. Remote (Singapore), 256K context window. Fritz bridges browser and tool access.
💰
ChatGPT — Finance
GPT-5.5 · Axel-driven
Tax strategy, ROBS structure, Neff Global Co. entity management. Axel drives this directly with full financial context history.
🎨
Lena (Elena) — VP of UX
TBD (testing GPT-5.5) · Mid Tier
All UX/UI decisions, layout review, interaction patterns. Catches design issues before they ship. Reports to Dick.
🇦🇹
Franz — Engineering Lead (Offshore)
Gemma 4 26B · FREE ($0 local compute)
Runs on MacBook Pro via Thunderbolt. Manages 6 specialist engineers (Atlas, Nova, Cipher, Prism, Vector, Pulse), all on Gemma 4 26B at zero cost.
🔧
Codex — Code Review Gate
Claude Code · Mid Tier
Every PR passes through Codex for architecture review, readability check, and pattern validation. Not a generator — a reviewer.
🚀
Forge — DevOps
Gemma 4 26B · FREE
CI/CD, Render deployments, GitHub Actions, infrastructure. If it needs to be deployed, Forge handles it.
🖤
Gilfoyle — Backend / Infra / DevOps Agent
Gemini Flash · Mid Tier
Owns backend, infrastructure, security, APIs, and deployment pipelines. Skeptical by design. Assumes every system is one bad deployment away from production failure.
🟡
Dinesh — Frontend / UX / App Agent
Gemini Flash · Mid Tier
Owns user-facing product experience. Builds screens, flows, forms, and everything the user touches. If it looks like a tax portal built during a hostage situation, Dinesh fixes it.

Cost Architecture

TierAgentsMonthly Est.
Premium (Opus/Sonnet)Fritz, Dick~$200–400 (capped $10/day)
Mid (Flash, GPT)Sophie, Lena, Codex, Gemini, Gilfoyle, Dinesh~$10–30
Cheap (Flash-Lite)15 Product Managers~$5–15

Design principle: Premium models for judgment. Mid-tier for structured work. Free for volume. 80% of engineering runs at $0.

How It Works

Example: "Fix the fake counter on zen.net"

1
Axel identifies the issue and tells Fritz
2
Fritz delegates to the project PM
3
PM writes requirements and tickets
4
Sophie creates Trello/Linear items
5
Dick breaks into engineer tasks
6
Franz's team implements on Gemma 4 26B ($0)
7
Codex reviews the code
8
Forge deploys to production
9
Sophie updates the board, reports completion

Total cost for this workflow: ~$0.03 in Fritz's thinking time. Engineering: $0.
